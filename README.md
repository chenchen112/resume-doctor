# Resume Doctor

简历诊断与优化工具，专为软件开发者设计。

## 功能

- **五步分析** — 整体评分、亮点识别、减分项警告、优化建议、ATS关键词
- **优化简历** — 基于分析生成高质量简历
- **面试预测** — 八股文 + 项目深挖 + 系统设计

## 支持岗位

前端 / 后端 / 算法 / 数据 / DevOps

## 输入

至少包含：岗位方向、工作年限、工作经历、项目经验（至少1个）

方式：粘贴文本、上传文件(.pdf/.docx/.txt/.md)、描述经历

## 输出示例

### 1. 整体评分 (10分制)

| 分数 | 标准 |
|------|------|
| 9-10 | 信息完整、数据量化、几乎无减分项 |
| 7-8 | 质量较高，少量可改进 |
| 5-6 | 明显问题（无量化、技术栈模糊、描述笼统） |
| 3-4 | 多个高危减分项 |

### 2. 亮点识别

根据岗位识别技术亮点（性能优化、高并发、架构设计等）

### 3. 减分项警告

识别模糊描述("负责/参与")、无量化数据、技术栈模糊等

### 4. 优化建议

每个减分项给出具体修改示例

### 5. ATS关键词分析

输出岗位匹配度分析和建议关键词

## 语言

- 中文简历 → 中文输出
- 英文简历 → 英文输出

## 触发方式

在 Claude Code 中描述你的简历或上传文件，即可获得分析
